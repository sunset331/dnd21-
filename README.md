## ✦ 这是什么？

一个 **DND 跑团风格的桌面宠物**——一颗 21 面骰子悬浮在你的桌面上，会呼吸、会浮动、会说骚话。

但它不只是玩具。**你的每一次掷骰真的会改变世界。**

偷了龙蛋？龙族会追杀你。帮了哥布林王？整个部落跟你称兄道弟。卖过黑市物品？地下商人给你打折。

## ✦ 核心系统

### 🌍 世界状态（v1.1.0 新增）

你的行为会留下**永久后果**：

| 系统 | 说明 |
|------|------|
| **背包** (`inventory`) | 获得的物品：龙蛋、黑市通行证、幽灵罗盘、星落剑…… |
| **声望** (`reputation`) | 阵营好感度：龙族 -50、哥布林 +30、暗精灵 +35…… |
| **标记** (`flags`) | 世界记忆：`stole_egg: true`、`dragon_blessed: true`…… |
| **叙事线** (`activeThreads`) | 连锁事件：偷蛋 → 黑市 → 龙骑追捕 → 龙骑结盟 |

### 📜 25 个故事事件

每个事件绑定 **4 档因果结局** + **世界后果**：

```
事件：偷龙蛋
├─ 大失败：幼龙破壳尖叫 → 母龙追猎
├─ 失败：被龙标记气味 → 龙族记住你
├─ 成功：带着龙蛋逃走 → 背包+龙蛋，声望-30，解锁黑市
└─ 大成功：龙蛋认主 → 母龙默许，声望+25
```

**10 条事件链**：偷龙蛋 → 黑市交易 → 龙骑追捕 → 龙骑结盟。你的选择会串成故事。

### 🧩 模块化事件引擎

当故事事件未触发时，组合引擎接管：

- **36 种行为** × **36 个地点** × **35 个目标** × **25 种意外** → 千万级组合
- 涵盖：暗精灵、半兽人、传奇幽灵、龙族、矮人、哥布林、天气系统、诅咒物品、阵营冲突
- 四档结果模板函数，根据事件组件动态生成因果文本

### 🎭 骰子情绪

骰子有 7 种情绪（兴奋/嘲讽/愤怒/无聊/中立/得意/困倦），会根据你的表现**变化辉光颜色**和**浮动速度**。长时间不点它会抱怨"地下城长蘑菇了"，深夜会低语"地下城在呼唤你"。

### 🛡️ 角色状态

冒险中获得 **醉酒、诅咒、幸运、哥布林气味、地牢通缉、圣光庇护** 等状态，影响后续事件概率。

### 🏆 成就系统

13 个搞怪成就：从"欢迎来到地牢"到"天选之人"，分 common / uncommon / rare 三档稀有度。

### 🎨 主题系统（v1.1.1 新增）

6 个预设主题一键切换：魔法紫 / 哥布林绿 / 龙焰橙 / 深海蓝 / 血月红 / 羊皮纸。每个主题可在设置面板中微调 5 种关键颜色，实时预览无需重启。调"主文字"颜色时，场景旁白自动跟随变暗。

### 🖥️ DPI 适配（v1.1.1 新增）

窗口自动感知 Windows 显示缩放（100%/125%/150%/200%），面板使用 `clamp()` 响应式布局防止溢出，所有弹窗有 `max-height: 90vh` 保护。

### 👻 幽灵模式

闲置 15 秒后骰子自动半透明化。鼠标靠近恢复——像漂浮的魔法遗物。

### 📜 冒险日志

完整记录每次掷骰：时间、事件、骰值、判定、结局。支持搜索和清空，最多 500 条，本地持久化。

## ✦ 安装与运行

### 方式一：便携版（推荐）

下载 `DND-Dice-Pet-v1.1.1.zip`，解压后双击 `DND Dice Pet.exe`。

### 方式二：从源码运行

```bash
git clone git@github.com:sunset331/dnd21-.git
cd dnd21-
npm install
npm start
```

> 中国大陆网络需配置镜像：`ELECTRON_MIRROR="https://npmmirror.com/mirrors/electron/" npm install`

## ✦ 项目结构

```
dnd-dice-pet/
├── main.js                  # Electron 主进程：透明置顶窗口 + 托盘
├── preload.js               # 安全 IPC 桥接
├── src/
│   ├── renderer/
│   │   ├── index.html       # SVG 水晶骰子 + UI
│   │   ├── app.js           # 核心协调器 + worldState
│   │   ├── dice.js          # 骰子状态机
│   │   ├── scenario.js      # 模块化事件引擎 + 25个故事事件
│   │   ├── status.js        # 角色状态管理
│   │   ├── mood.js          # 骰子情绪引擎
│   │   ├── ghost.js         # 幽灵模式
│   │   ├── achievements.js  # 成就系统
│   │   ├── adventure-log.js # 冒险日志
│   │   ├── context-menu.js  # 右键菜单
│   │   ├── settings.js      # 设置面板 + 持久化
│   │   ├── ui-manager.js    # 弹窗单例管理
│   │   ├── audio.js         # Web Audio 过程化音效
│   │   ├── animation.js     # 粒子特效
│   │   └── result-popup.js  # 判定弹窗
│   └── styles/              # 7个CSS文件
└── .tours/                  # CodeTour 代码导览
```

**零框架** — 纯 HTML/CSS/JS + Electron，15 个 JS 模块。
