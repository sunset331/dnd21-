# DND 桌宠 — 旅行青蛙 × DND 跑团

> 挂机养成桌面 RPG 伴侣 | Electron + HTML/CSS/JS

一个可爱的 Q 版奇幻冒险者住在你的桌面上。它会自动感知你的键盘鼠标活动，每积累 300 次操作就触发一次 D20 骰子判定冒险。赚金币、买装备、换皮肤、触发奇遇——一切都在桌面角落悄然发生。

## 核心玩法

- **挂机冒险**：自动追踪键盘鼠标操作 → 每 300 次触发 D20 冒险
- **D20 骰子**：1=大失败 / 2-5=失败 / 6-10=小成功 / 11-15=成功 / 16-19=大成功 / 20=传奇
- **7 槽位装备**：帽子·武器·戒指·项链·护手·披风·鞋子，全部纸娃娃实时显示
- **10 场景背景**：酒馆·洞穴·神殿·悬崖·森林·沼泽·龙巢·广场·古墓·巫师塔
- **5 种族**：人类·精灵·矮人·哥布林·龙裔
- **23 个冒险事件**：含市集失窃·悬崖遇险·酒馆斗殴·诅咒遗物·沉船逃生
- **商城系统**：35 件装备（普通→传说），每日特价 8 折
- **成就系统**：16 个成就，5 大类别
- **3 槽位存档**：独立角色互不覆盖，随时切换
- **自动模式**：右键开启，每 60 秒自动冒险，无需操作

## 技术栈

- **框架**：Electron 33
- **渲染**：Canvas 2D + CSS 3D
- **生图**：ComfyUI + SDXL（本地 API）
- **动画**：requestAnimationFrame 驱动，4 阶段缓动序列
- **存档**：JSON 多槽位 + 轮转备份

## 快速开始

```bash
npm install
npm start
```

首次启动自动弹出角色创建界面。

右键窗口打开菜单：自动冒险 / 面板 / 改名 / 新建角色 / 切换角色 / 删除角色。
